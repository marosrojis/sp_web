Obsah jednotlivých adresářů (viz dokumentace):

app\config\                                     konfigurační soubory
src\Autodoprava\AppBundle\Controller\           kontrolery
src\Autodoprava\AppBundle\Form\                 formuláře
src\Autodoprava\AppBundle\Model\                modely pro práci s databází
src\Autodoprava\AppBundle\Resources\config\     konfigurační soubory pro nastavení adres a služeb
src\Autodoprava\AppBundle\Resources\views\      šablony
src\Autodoprava\AppBundle\Security\User\        třídy pro zabezpečené přihlášení do redakčního systému
web\                                            základní soubory webu
web\bundles\app\css\                            CSS soubory
web\bundles\app\img\                            obrázky pro design stránek
web\bundles\app\js\                             JavaScript soubory
